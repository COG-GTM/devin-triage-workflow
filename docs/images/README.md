## Adding Custom Screenshots

For the best customer experience, you can add your own screenshots of:

### Azure Portal
1. **Action Groups list** - Azure Portal → Monitor → Alerts → Action Groups
2. **Create Action Group wizard** - Each tab (Basics, Notifications, Actions)
3. **Webhook configuration** - Action type dropdown → Webhook settings
4. **Alert Rules list** - Azure Portal → Monitor → Alerts → Alert Rules
5. **Create Alert Rule wizard** - Scope, Condition, Actions tabs

### Devin App
1. **Settings page** - [app.devin.ai/settings](https://app.devin.ai/settings)
2. **API Keys section** - [app.devin.ai/settings/api-keys](https://app.devin.ai/settings/api-keys)
3. **Session view** - Any active Devin session

### Kibana
1. **Connectors list** - Stack Management → Connectors
2. **Webhook connector form** - Create connector → Webhook
3. **Alerting rules** - Stack Management → Rules
4. **Rule creation** - New rule wizard

## Screenshot Guidelines

- Use **PNG** for UI screenshots (crisp text)
- Use **JPG** for documentation pages (smaller file size)
- Recommended width: **800-1200px**
- Redact any sensitive information (API keys, internal URLs)
